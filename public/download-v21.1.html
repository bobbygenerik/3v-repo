🚘 Tres3 v21.1 — Bluetooth Car Mode + Audio

What’s new

  • Prefer Bluetooth route for calls when available:
    • API 31+: setCommunicationDevice(BLUETOOTH_SCO)
    • Older devices: startBluetoothSco fallback
  • Don’t force speakerphone if Bluetooth is present.
  • Stronger audio focus request (voice communication) so music apps pause.
  • Clean up on hangup: clear communication device, stop SCO, abandon focus.
  • Incoming-call mic/camera permission gate (fixes “remote can’t hear me”).
