## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd /repos/tres3/3v-repo/tres_flutter
flutter pub get
```

### 2. Run Firebase Setup Script (15 minutes)
```bash
./setup_firebase.sh
```

This will:
- Install Firebase CLI and FlutterFire CLI
- Login to Firebase
- Configure your Firebase project
- Download config files (google-services.json, GoogleService-Info.plist)
- Update environment.dart with your project ID

### 3. Set Up LiveKit (10 minutes)
**Option A - LiveKit Cloud (Recommended):**
1. Go to https://cloud.livekit.io/
2. Sign up (free tier: 10K minutes/month)
3. Create project → Copy API Key, Secret, URL

**Option B - Self-hosted:**
```bash
docker run -d -p 7880:7880 livekit/livekit-server --dev
```

### 4. Configure Backend (5 minutes)
```bash
cd ../functions
cp .env.example .env
# Edit .env with Firebase & LiveKit credentials
npm install
firebase deploy --only functions
```

### 5. Run the App!
```bash
cd ../tres_flutter
flutter run
```

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| "Firebase not initialized" | Run `./setup_firebase.sh` or `flutterfire configure` |
| "LiveKit connection failed" | Check WebSocket URL format (`wss://` not `https://`) |
| "ML Kit not working" | Use real device, not emulator |
| "Functions timeout" | Check `firebase functions:log` and verify `.env` deployed |
| "Permission denied" | Grant camera/microphone permissions in device settings |
