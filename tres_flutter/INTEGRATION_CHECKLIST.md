## 🔄 Setup Required (Do Once)

### 1. Firebase Project Setup (15 minutes)

**Run the automated script:**
```bash
cd /repos/tres3/3v-repo/tres_flutter
./setup_firebase.sh
```

**Manual steps if needed:**
```bash
# Install tools
dart pub global activate flutterfire_cli
npm install -g firebase-tools

# Login and configure
firebase login
flutterfire configure \
  --project=YOUR_PROJECT_ID \
  --platforms=android,ios,web \
  --android-package-name=com.threeveesocial.tresvideo \
  --ios-bundle-id=com.threeveesocial.tresvideo
```

**Enable Firebase services in console:**
- ☐ Authentication (Email/Password, Google, Anonymous)
- ☐ Firestore Database
- ☐ Storage
- ☐ Cloud Messaging

### 2. LiveKit Setup (10 minutes)

**Option A - LiveKit Cloud (Recommended):**
1. Go to https://cloud.livekit.io/
2. Create account (free tier: 10K minutes/month)
3. Create project
4. Copy API Key, Secret, URL

**Option B - Self-Hosted:**
```bash
docker run -d -p 7880:7880 -p 7881:7881 -p 7882:7882/udp \
  livekit/livekit-server --dev
```

### 3. Backend Functions Setup (10 minutes)

```bash
cd /repos/tres3/3v-repo/functions

# Create .env from template
cp .env.example .env

# Edit with your credentials
nano .env

# Required values:
# - FIREBASE_PROJECT_ID
# - FIREBASE_CLIENT_EMAIL
# - FIREBASE_PRIVATE_KEY
# - LIVEKIT_API_KEY
# - LIVEKIT_API_SECRET
# - LIVEKIT_URL

# Deploy
npm install
firebase deploy --only functions
```

### 4. Update Flutter Configuration (5 minutes)

Edit `lib/config/environment.dart`:
```dart
static const String liveKitUrl = 'wss://your-actual-server.livekit.cloud';
static const String functionsBaseUrl = 
    'https://us-central1-YOUR_ACTUAL_PROJECT_ID.cloudfunctions.net';
```

Also update `android/app/src/main/AndroidManifest.xml`:
```xml
<!-- Replace in deep link section -->
<data
    android:scheme="https"
    android:host="YOUR_ACTUAL_PROJECT.web.app"
    android:pathPrefix="/join"/>
```

### 5. Install Dependencies (2 minutes)

```bash
cd /repos/tres3/3v-repo/tres_flutter
flutter pub get
```
